## 🔑 Enable Real Groq API (15 Minutes)

### Getting Your API Key

```bash
# 1. Visit https://console.groq.com
# 2. Create account (free, no credit card required)
# 3. Navigate to API Keys
# 4. Create new key
# 5. Copy the key (starts with gsk_)
```

### Configuration

```bash
# Edit backend/.env
GROQ_API_KEY=gsk_YOUR_KEY_HERE
GROQ_MODEL=llama-3.1-8b-instant
LLM_PROVIDER=groq
```

### Activate

```bash
# Stop services
pkill -f "uvicorn"
pkill -f "celery"

# Restart backend
cd backend
python -m uvicorn app.main:app --reload --port 8000 &

# Restart Celery
celery -A app.workers.celery_app worker --loglevel=info &

# Verify real API calls are being made
# Monitor at https://console.groq.com/usage
```

## 🎯 How It Works

### Architecture
```
User Code → FastAPI → Celery → LLM Generator → Groq API → Test Code → Database
                                  (or Demo)
```

### Test Generation Process
1. **Extract** - Find functions in your code
2. **Analyze** - Identify edge cases and patterns
3. **Prompt** - Create AI-optimized prompt with context
4. **Generate** - Call Groq API (or use demo) for test code
5. **Store** - Save results with metadata
6. **Display** - Show in frontend dashboard

### Example

**Input:**
```python
def divide(a, b):
    if b == 0:
        raise ValueError("Cannot divide")
    return a / b
```

**Generated Test (via Groq):**
```python
def test_divide_by_zero():
    """Test divide raises error for zero divisor"""
    with pytest.raises(ValueError, match="Cannot divide"):
        divide(10, 0)

def test_divide_normal():
    """Test divide with positive numbers"""
    assert divide(10, 2) == 5.0
```
